3.  atualizar arquivo no git
3.1 após alteração, fazer um $ git status para visualizar as alterações
3.2 em seguida fazer um $ git add <arquivo> ou $ git add . para atualizar todos os arquivos
3.3 $ git commit -m "<nome_do_commit>"
3.4 $ git push origin master para atualizar o repositório origin com as alterações realizadas em master
